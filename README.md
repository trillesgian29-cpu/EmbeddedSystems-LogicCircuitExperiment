# EmbeddedSystems-LogicCircuitExperiment

## 📘 Project Overview
This repository documents my laboratory experiment in Computer Engineering, where the **only given input was a Boolean expression**. Based on that, I constructed the truth table, designed the logic circuit, simulated it in AutoCAD, simplified it using Karnaugh Maps (K-map), converted the design into NAND-only implementation, and finally tested the circuit using ICs in the laboratory.

---

## 🛠️ Workflow

### 1. Boolean Expression
- Instructor provided a Boolean expression as the starting point.
- This served as the foundation for all subsequent steps.

### 2. Truth Table (SQ1)
- Constructed a complete truth table to verify all possible input/output combinations.
- Ensured logical consistency with the given Boolean expression.
- 📂 See: `truth-table/SQ1.png`

### 3. Logic Gate Design (SQ2)
- Drafted the circuit using basic logic gates (AND, OR, NOT).
- Created schematic diagrams in AutoCAD for clarity.
- Produced both draft and final versions of the design.
- 📂 See: `logic-gates-design/SQ2.png`

### 4. K-Map Simplification
- Applied Karnaugh Map techniques to minimize the Boolean expression.
- Reduced the number of gates required, optimizing the design.
- 📂 See: `kmap/Kmap.png`

### 5. NAND Conversion (SLC)
- Converted the simplified circuit into NAND-only implementation.
- Demonstrated universality of NAND gates in digital logic.
- 📂 See: `nand-conversion/SLC.png`

### 6. Final Hardware Implementation
- Built the circuit using 7400 series ICs (AND, OR, NOT, NAND).
- Tested on breadboard with power supply and LEDs.
- Verified outputs matched the truth table.
